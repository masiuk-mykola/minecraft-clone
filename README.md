## Minecraft Clone (React + Three.js)

A small first‑person voxel prototype built with React, Three.js, and React Three Fiber. It includes physics via Cannon and basic movement controls.

### Features

- First‑person camera with pointer lock
- Physics‑based player movement
- Ground plane with tiled texture
- Texture selection hotkeys (1–5)

### Controls

- Move: W / A / S / D
- Jump: Space
- Select texture: 1–5

### Getting Started

Install dependencies:

```bash
npm install
```

Run the dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

### Tech Stack

- React + TypeScript
- Three.js + React Three Fiber
- @react-three/cannon for physics
- Zustand for state
